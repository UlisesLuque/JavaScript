¿Qué es Local Storage?

Con local storage, las aplicaciones web pueden almacenar datos localmente sin el navegador del usuario.

A diferencia de las cookies que permiten almacenar 4KB, el límite de almacenamiento de Local Storage es de 5MB. Además las cookies tienen fecha de expiración, explícita o implícita, mientras que Local Storage no tiene fecha expiración.

Para almacenar un dato en Local Storage se utiliza el método setItem: localStorage.setItem("apellido", "Pérez");

También se puede utilizar la siguiente sintaxis: localStorage.apellido = "Pérez";

Para eliminar un dato de Local Storage se utliza el método removeItem: localStorage.removeItem("apellido");

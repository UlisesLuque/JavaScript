¿Que es Local Storage?

Con local storage, las aplicaciones web pueden almacenar datos localmente sin el navegador del usuario.

A diferencia de las cookies que permiten almacenar 4KB, el limite de almacenamiento de Local Storage es de 5MB. Además las cookies tienen fecha de expiración, explicita o implicita, mientras que Local Storage no tiene fecha expiración.

Para almacenar un dato en Local Storage se utiliza el método setItem: localStorage.setItem("apellido", "Luque");

También se puede utilizar la siguiente sintaxis: localStorage.lastname = "Smith";

Para eliminar un dato de Local Storage se utliza el método removeItem: localStorage.removeItem("apellido");

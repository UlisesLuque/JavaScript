¿Qué es Session Storage?

Con Session Storage, las aplicaciones web pueden almacenar datos localmente sin el navegador del usuario.

La única diferencia entre Local Storage y Session Storage es que en Local Storage, los datos son almacenados solo durante la sesión. Los datos son eliminados cuando se cierra la pestaña del navegador.

Para almacenar un dato en Session Storage se utiliza el método setItem: sessionStorage.setItem("apellido", "Luque");

También se puede utilizar la siguiente sintaxis: sessionStorage.lastname = "Smith";

Para eliminar un dato de Session Storage se utliza el método removeItem: sessionStorage.removeItem("apellido");

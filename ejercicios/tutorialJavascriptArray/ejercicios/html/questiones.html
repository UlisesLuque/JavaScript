  1. pop() y push(),
    1. Utilidad

EL método pop() eliminar el último elemento del array y lo devuelve.

El método push() añade un elemento en la última posición del array.

    1. ¿Modifican el array original?

Ambos modifican el array el original.

    1. ¿Devuelven algún valor?

El método pop() devuelve el elemento eliminado del array.

El método push() no devuelve nada.

  1. shift() y unshift(),
    1. Utilidad

El método shift() elimina el primer elemento del array y lo devuelve.

El método unshift() introduce un elemento en la primera posición de la array.

    1. ¿Modifican el array original?

Ambos modifican el array original.

    1. ¿Devuelven algún valor?

El método shift()  devuelve el elemento eliminado.

El método unshift() no devuelve nada.

  1. De los cuatro métodos anteriores, indica los recomendados para la inserción/eliminación de un elemento y por qué.

Depende del índice del elemento sobre el que queremos actuar. Pero al refactorizar los índices, los métodos push() y pop() son más rapidos que shift() y unshift().

  1. join() y split(), ¿son métodos de String o de Array?

El método split() pertenece a String, pero convierte la variable en un array.

El método join() pertenece a Array, pero convierte la variable en un String.
